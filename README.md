# Estandar Max&Min BOH - v4 Ajuste Imagen PDF

PWA ejecutiva para documentar evidencias de orden, limpieza, máximos y mínimos en Back of House.

## Cambios v4
- Ajuste definitivo de fotos ANTES / DESPUÉS en PDF con lógica tipo LayOut.
- Las imágenes llenan el recuadro asignado con proporción conservada y clip centrado.
- Nombre de archivo simplificado: `Estandar_Max&Min_[CeCo]_[Fecha].pdf`.
- Cache PWA actualizado.

## Estructura
- `index.html` en raíz.
- `app.js` generación PDF y flujo operativo.
- `styles.css` estilos UI.
- `manifest.webmanifest` PWA.
- `sw.js` cache offline.
- `.nojekyll` para GitHub Pages.
- `assets/` iconos e imágenes de referencia.

Publicar desde GitHub Pages: `main / root`.
